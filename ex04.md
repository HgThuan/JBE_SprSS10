# Định dạng dữ liệu trả về thống nhất (Response Wrapper)

Khi thiết kế RESTful API, việc trả về một cấu trúc JSON đồng nhất cho mọi API giúp phía Frontend dễ dàng bắt và xử lý thành công, lỗi, hoặc thông tin phân trang (pagination) mà không cần phải viết lại logic quá nhiều lần.

### 1. Mẫu JSON trả về cho API "Lấy danh sách bác sĩ"

Dưới đây là một ví dụ mẫu khi trả về danh sách bác sĩ:

```json
{
  "status": "success",
  "code": 200,
  "data": [
    {
      "id": 1,
      "name": "Nguyen Van A",
      "specialty": "Tim mạch"
    },
    {
      "id": 2,
      "name": "Tran Thi B",
      "specialty": "Thần kinh"
    }
  ],
  "meta": {
    "totalRecords": 50,
    "currentPage": 1,
    "totalPages": 5,
    "pageSize": 10
  }
}
```

### 2. Thực hành: Xây dựng class Java `ApiResponse<T>`

Sử dụng Generic (`<T>`) trong Java cho phép class này bao bọc bất kỳ loại dữ liệu nào vào trường `data` (một bác sĩ, danh sách bác sĩ, hay bất kỳ model nào khác).

```java
import com.fasterxml.jackson.annotation.JsonInclude;

// Bỏ qua các trường có giá trị null khi parse sang JSON (ví dụ nếu không có meta, nó sẽ không hiện field meta)
@JsonInclude(JsonInclude.Include.NON_NULL)
public class ApiResponse<T> {
    private String status;
    private int code;
    private T data;
    private Object meta; // Có thể định nghĩa một class PageMeta riêng cho field này

    // Constructor mặc định
    public ApiResponse() {}

    // Constructor dùng cho trả lời lỗi (không có data)
    public ApiResponse(int code, String status) {
        this.code = code;
        this.status = status;
    }

    // Constructor dùng khi trả về dữ liệu (không có phân trang)
    public ApiResponse(int code, String status, T data) {
        this.code = code;
        this.status = status;
        this.data = data;
    }

    // Constructor dùng khi trả về danh sách kèm phân trang (meta)
    public ApiResponse(int code, String status, T data, Object meta) {
        this.code = code;
        this.status = status;
        this.data = data;
        this.meta = meta;
    }

    // --- Getter và Setter ---
    
    public String getStatus() {
        return status;
    }

    public void setStatus(String status) {
        this.status = status;
    }

    public int getCode() {
        return code;
    }

    public void setCode(int code) {
        this.code = code;
    }

    public T getData() {
        return data;
    }

    public void setData(T data) {
        this.data = data;
    }

    public Object getMeta() {
        return meta;
    }

    public void setMeta(Object meta) {
        this.meta = meta;
    }
}
```
